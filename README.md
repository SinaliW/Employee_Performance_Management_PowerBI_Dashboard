# Employee_Performance_Management_PowerBI_Dashboard

## Project Overview
The **Employee Performance Management Dashboard** is an interactive Power BI solution designed to help HR managers and business leaders monitor employee performance, attendance, training effectiveness, and appraisal trends across the organization.

The dashboard integrates **SQL**, **Power BI**, and **DAX** to transform employee performance data into actionable insights through interactive visualizations, KPIs, and drill-through analysis.

## Objectives
- Monitor employee performance across departments and job levels.
- Track attendance and training effectiveness.
- Analyze performance trends over time.
- Identify high-performing and underperforming departments.
- Explore factors influencing employee performance.
- Provide detailed employee-level performance reviews.
